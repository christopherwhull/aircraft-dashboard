# Stashed Files

This directory contains files that have been moved out of the active codebase but preserved for reference.

## PowerShell Scripts (`powershell-scripts/`)

All PowerShell scripts have been moved here as they have been replaced with Python equivalents for cross-platform compatibility. The scripts include:

- `capture-screenshots.ps1` - Screenshot capture utility
- `clear-tile-cache.ps1` - Tile cache clearing tool
- `publish-wiki.ps1` - Wiki publishing script
- `reproject-charts.ps1` - Chart reprojection utility
- `restart-node-server.ps1` - Node server restart script
- `restart-server-headless.ps1` - Headless server restart
- `restart-server.ps1` - Server restart utility
- `restart_server.ps1` - Alternative server restart script
- `run-tests.ps1` - Test runner (replaced by `run_tests.py`)
- `start-proxy-3004.ps1` - Tile proxy starter (replaced by `start_servers.py`)
- `start_all_terminals.ps1` - Terminal launcher for all services
- `test-all.ps1` - Test suite (replaced by `test_all.py`)
- `test-endpoints.ps1` - Endpoint testing (replaced by `test_endpoints.py`)
- `trigger-pages-rebuild.ps1` - Pages rebuild trigger

## Other Files

- `aircraft_tracker.py` - Previous version of the aircraft tracker
- `geotiff-cache-status.test.js` - GeoTIFF cache testing utility
- `geotiff-server.js` - GeoTIFF server implementation

These files are preserved for historical reference but are not part of the active codebase.
