# Release v1.4.1 (2025-12-02)

### Fixed
- **Tooltip field overwrites**: Prevented batch/lightweight flight responses from overwriting populated live fields (registration, altitude, speed, airline) by using a defensive merge helper `mergePreferExisting` in `public/heatmap-leaflet.html`.
- **Squawk propagation & hover wait**: Added `fetchSquawkWithWait()` to poll `/api/squawk` for up to 1s on hover and update tooltips when squawk becomes available.

### Added
- **Headless tooltip validation**: `tools/headless-tooltip-test.js` updated to validate every visible marker tooltip and verify the `/api/squawk` endpoint; added `test:headless` npm script.
- **Screenshot helper**: `tools/screenshot-heatmap.js` to capture `heatmap-leaflet.html` screenshots for documentation (saves to `docs/screenshots/`).

### Docs
- Documented changes and added `docs/HTML_PAGES.md` with a screenshot of the Leaflet heatmap page.
