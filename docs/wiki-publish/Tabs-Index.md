# Tabs Index

This page lists the UI tab pages with thumbnails for quick reference.

- **Tab: Live** — [Live](Live)
  The Live tab shows real-time aircraft tracking, statistics and a live aircraft table.

  ![Live tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-live.png)

- **Tab: Airlines** — [Airlines](Airlines)
  The Airlines tab provides per-airline statistics and drilldowns for selected time ranges.

  ![Airlines tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-airlines.png)

- **Tab: Flights** — [Flights](Flights)
  The Flights tab reconstructs and lists historical flights with detailed flight metadata.

  ![Flights tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-flights.png)

- **Tab: Positions** — [Positions](Positions)
  The Positions tab shows time-series statistics and charts for position data from multiple sources.

  ![Positions tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-positions.png)

- **Tab: Squawk** — [Squawk](Squawk)
  The Squawk tab analyzes squawk code changes and categorizes transitions (VFR/IFR/emergencies).

  ![Squawk tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-squawk.png)

- **Tab: Heatmap** — [Heatmap](Heatmap)
  The Heatmap tab provides density visualizations and an external Leaflet viewer for large datasets.

  ![Heatmap tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-heatmap.png)

- **Tab: Reception** — [Reception](Reception)
  The Reception tab analyzes receiver coverage by bearing, altitude and 3D reception range plots.

  ![Reception tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-reception.png)

- **Tab: Cache Status** — [Cache Status](Cache-Status)
  The Cache Status tab shows the server's cache summaries including types DB and position cache metrics.

  ![Cache tab](https://raw.githubusercontent.com/christopherwhull/aircraft-dashboard/main/docs/screenshots/index-cache.png)
